Key Solutions & Innovations

Running real-time image evaluation on a mobile SoC is highly resource-constrained. Standard CNN-LSTM models require high GPU capabilities and memory bandwidth. To resolve this, I compressed the custom-trained weights and loaded a serialized flatbuffer representation using the TensorFlow Lite Interpreter API.

By implementing asynchronous channel streams, frame acquisition runs in a separate native worker thread, allowing the main rendering thread to run smooth Jetpack Compose UI loops without UI frame drops.

Case Study: ArtListener

ArtListener is a context-aware museum audio guide application designed to automatically fetch and play descriptions of nearby museum exhibits based on indoor position detection signals.

Key Solutions & Innovations

GPS signals do not penetrate building interiors, making them unusable for indoor localization. Standard Bluetooth Beacon solutions require expensive dedicated hardware installation. ArtListener circumvents this by scanning existing institutional Wi-Fi networks.

By measuring the Received Signal Strength Indicator (RSSI) of exhibit-adjacent access points and applying a logarithmic distance attenuation formula, the application calculates proximity bounds within a 1.5-meter accuracy window, triggering the correct audio guides synchronously.
